Split/Second — Reverse-Engineering Wiki

An independent, fan-made wiki documenting the internals of Split/Second (2010, Black Rock Studio / Disney Interactive) — its engine architecture, its ~70 binary file formats, and its game systems. Everything here is reconstructed from a devkit/QA debug build (title id NPXX00575), whose loose, uncooked data files and symbol-rich executable make the engine unusually legible.

This documents the Black Rock engine — it is not Burnout. Where this wiki resembles the Burnout wiki, that's house style, not shared technology.

How this wiki is organised

  • Overview & Engine — architecture, rendering, threading, audio, networking, resources.
  • Data & Config — the .params tuning system, the global registry, and the Catnip entity system.
  • File Formats — one page per format, grouped into Geometry/Render, Physics (Havok), Audio/Video, FX, Gameplay, Track/Route, Archive, and Telemetry.
  • Game Systems — game modes, vehicles, cameras, scoring, tracks, difficulty/race-director, HUD, and the boss modes.

Conventions

Format pages carry a confidence badge: Confirmed (verified against data, XML twins, or the executable), Partial (header understood, some fields open), Theory (reasoned guess), and Opaque (third-party/middleware blob whose structure is defined elsewhere). Offsets and lengths are hexadecimal; all multi-byte values are big-endian (PS3 PowerPC) unless noted.
