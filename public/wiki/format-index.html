All File Formats — Master Index

Every distinct file extension found under USRDIR/Deferred in the devkit build (15,703 files, ~5.2 GB, ~66 extensions), with file counts, on-disk totals, a one-line role, and a link to the page that documents it. Use this as the map; click an extension to jump to its full write-up. See Build & File Layout for the directory tree.

Sizes are on-disk totals for the loose (uncooked) devkit data. All binary formats are big-endian (PS3 PowerPC). Confidence badges live on each format's own page.
